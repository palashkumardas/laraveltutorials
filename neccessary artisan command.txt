 necessary artisan command
-------------------------------------

1. php artisan serve				 = To run laravel application

2. php artisan make:Controller Controllername	 = To make new controller

3. php artisan make:Model Modelname		 = To make new model
 
4. php artisan make:Middleware Middlewarename	 = To make middleware class

5. php artisan make:migration tablename		 = To make migration file

6. php artisan migrate				 = To run migration


7. php artisan cache:clear			= clear application cache

8. php artisan route:clear			= clear route cache

9. php artisan config:clear			= clear config cache

10. php artisan view:clear			= clear compiled view files
